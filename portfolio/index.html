FIS

eZCardInfo.com

This project was the total redesign of a white-label banking application with millions of users. The old site was 10+ years old and in dire need of an update. We used client and user feedback to create two design templates, both of them responsive. A separate admin application allows the admins to configure settings, access, themes and custom colors. It was built using Angular and C#. The CSS code base was build completely from scratch and is used in conjunction with Bootstrap.

Tools and software: LESS, Angular, Visual Studio, TFS, and SMACSS principles
